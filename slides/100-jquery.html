100% jQuery

Remember those 5 steps we went through previously? This jQuery function will handle all of that. At once. In one function.

$.ajax({type: "GET",
  url: "books.json",
  dataType: "json",
  success: function(books) {
    for (var i = 0; i < books.length; i++) {
      var book = books[i];
      var p = $('<p>');
      p.html(book.title + ' by ' + book.author);
      $('body').append(p);
    }
  }
});
