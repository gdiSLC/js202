AJAX + Servers

Instead of using $.ajax() with a file, you can use it with an actual server, too.

$.ajax({
  type: "GET",
  url: "/api/books",
  dataType: "json",
  data: {'sortBy': 'author'},
  success: processBooks
});

$.ajax({
  type: "POST",
  url: "/api/book/new",
  dataType: "json",
  data: {'title': 'I, Robot', 'author': 'I. Asimov'}
  success: processBoooks
});

This code won't really work - it's just an example.
