JS + JSON + DOM

We can use JavaSript to insert our JSON into the DOM.

var myProfile = {
  "firstName": "Liz",
  "lastName": "Howard",
  "cats": ["Tribbles", "Jean Claws"]
};

var p = document.createElement('p');
p.innerHTML = myProfile.firstName + ' ' + myProfile.lastName;
p.innerHTML += 'My cats are ' + myProfile.cats.join(',');
