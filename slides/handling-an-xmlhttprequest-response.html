(3) Handling Response

function processRequest() {
  if (request.readyState == 4) {
    if(request.status == 200) {
      processResponse(request);
    } else {
     alert("Error loading page " +
       request.status + ":" +
       request.statusText);
    }
  }
}

You can use the readyState property to check if the request has been completed, and status to check if it completed successfully.
